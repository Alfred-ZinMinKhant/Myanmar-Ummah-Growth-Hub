Myanmar Ummah Growth Hub
========================

A simple static single-file React site (Babel standalone) showcasing the Myanmar Ummah Growth Hub — hero, community gallery, events, and join flow.

Quick start
-----------
- Open `index.html` in a modern browser (no build step required).
- For local development with live reload, serve the folder with a static server, e.g.:

```bash
# with Python 3
python -m http.server 8080
# then open http://localhost:8080
```

Images
------
- Image assets live in the `assets/` folder.

Preview modal
-------------
- Click the hero or any gallery image to open an inline preview modal.
